# @uppercod/keen-slider

Es un wrapper para keen-slider como webcomponent creado con Atomico.

## Example

### HTML

```html
<keen-slide>
  <div>slide 1</div>
  <div>slide 2</div>
  <div>slide 3</div>
  <div>slide 4</div>
</keen-slide>
```

## Api

actualemnte el componente refleja las siguientes opciones del [api original](https://keen-slider.io/api/#api).

### centered

```html
<keen-slide centered>
  <div>slide 1</div>
  <div>slide 2</div>
  <div>slide 3</div>
  <div>slide 4</div>
</keen-slide>
```

### slidesPerView | slides-per-view

```html
<keen-slide slides-per-view="1, 2 720px 3 1080px">
  <div>slide 1</div>
  <div>slide 2</div>
  <div>slide 3</div>
  <div>slide 4</div>
</keen-slide>
```
